# CARA — Modulation of Photomorphogenesis in Spaceflight Influences Root System Architecture

OSDR Analysis Working Group manuscript + re-analysis for the CARA spaceflight
experiment. Published as a GitBook.

> Merge of the former `README.md` and `README (1).md`. Fill in / trim as needed.

## Contents

- **Manuscript chapters** — introduction, plant tropisms & microgravity,
  gravitropism, epigenetic map of the root tip, discussion, conclusion,
  references.
- **`methods/`** — root diameter / length / surface / volume physiology (ANOVA),
  GLARE summary.
- **`results/`** — GO & PPI/MCODE enrichment networks, root-skewing loci,
  Metascape skewing↔OSDR intersections, analysis reports. See
  [`results/README.md`](results/README.md).

## Data source & provenance

- NASA OSDR study: **OSD-120** (CARA / APEX root photography + RNA-seq).  *<- confirm/add accessions*
- Skewing loci: BMC Plant Biology 2017 (12870_2017_975).
